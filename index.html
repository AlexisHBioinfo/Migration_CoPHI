User guide for the system CoPHI.

CoPHI : Coordonnées Parallèles pour les réseaux métaboliques

CoPHI is an interactive visualization system of parallel coordinates.

Finding reaction patterns through metabolic pathways allows the identification of constraints and interactions between enzymatic reactions.

Computing Elementary Flux modes is a way to know feasible pathways through a metabolic network. The set size of these feasible pathways requires tools to display results. CoPHI is a user-friendly web application to display set of Elementary Flux modes using a parallel coordinates graph.

How to import data

To import data you have to click the "Parcourir..." button from the "Import data" panel. You can now choose the file to import from the new opened window. The file you select must be a table where values are split by comas, tabulations, spaces or semicolons.

Once the file is selected, its name is written next to the "Parcourir.. " button and you may click the "Submit" button generating a graph.

How to display set of data

To display the set of data you have imported you just have to click anywhere on "Data" bar of the main window. A table with the set of data imported is now displayed below.

How to zoom in or zoom out

To zoom in or zoom out the graph, you just have to click the word "Zoomer" and hover your mouse over the graph, then scroll up to zoom in or scroll down to zoom out. You may click again the word "Zoomer" to go back to the previous setting where scrolling doesn't zoom at all.

How to displace axes

To displace axes, you have to click the words "Déplacer les axes" which allows axes to be dragged where you want. You may click again the words "Déplacer les axes" to go back to the previous setting where clicks don't interact with the graph.

How to select and highlight

To highlight something, you have to click the words "Sélectionner aggregat et ses descendants" and select what you want to highlight. You may click again the words "Sélectionner aggregat et ses descendants" to go back to the previous setting where clicks don't interact with the graph.
