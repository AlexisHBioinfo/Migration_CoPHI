User guide for the CoPHI system

CoPHI is an interactive visualization system of parallel coordinates.

Finding reaction patterns through metabolic pathways allows the identification of constraints and interactions between enzymatic reactions.

Computing Elementary Flux modes is a way to know feasible pathways through a metabolic network. The set size of these feasible pathways requires tools to display results. CoPHI is a user-friendly web application to display sets of Elementary Flux modes using a parallel coordinates graph.

How to import data

To import data you have to click the "Select a file" button from the left panel.

You can now choose the file to import from the new opened window. The file you select must be a table where values are split by comas, tabulations, spaces or semicolons.

Once the file is selected you may click the "Submit" button so it generates a graph.

How to display a set of data

To display the set of data you have imported you have to click anywhere on the "Data" bar of the main window. A table with the set of imported data is now displayed below.

How to zoom in or zoom out

To zoom in or zoom out the graph, you have to click the first button of the top panel and hover your mouse over the graph, then scroll up to zoom in or scroll down to zoom out.

You may click again the first button of the top panel to go back to the previous setting where scrolling does not zoom at all.

How to displace axes

To displace axes, you have to click the fifth button of the top panel which allows axes to be dragged anywhere you want.

You may click again the fifth button of the top panel to go back to the previous setting where clicks do not interact with the graph.

How to change the color scheme

To change the color scheme, you have to hover the word "Color Scheme" and click on the scheme you want. If the graph is already loaded, you have to displace an axis to change the scheme. If the graph has not been loaded yet, you have to load it and it should have the good scheme.
